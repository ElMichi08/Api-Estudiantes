# Api-Estudiantes
API REST con SpringBoot, JDK 21, con FireBase

- Configuración inicial del proyecto Spring Boot.
- Implementación de CRUD para estudiantes con Firebase como base de datos.
- Endpoints:
  - POST /api/estudiantes: Crear estudiante.
  - GET /api/estudiantes: Obtener todos los estudiantes.
  - GET /api/estudiantes/{id}: Obtener un estudiante por ID.
  - DELETE /api/estudiantes/{id}: Eliminar un estudiante por ID.
- Configuración de .gitignore para excluir archivos sensibles como el archivo de credenciales Firebase.
- Listo para exponer la API a través de ngrok para pruebas externas.
